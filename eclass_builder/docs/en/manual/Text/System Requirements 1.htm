To run EClass.Builder and view EClasses, you need the following:

PC:
200Mhz+ Pentium or AMD x86 processor
64MB RAM
100 MB Hard drive space (over 1GB hard drive space recommended for storing EClasss)
Windows 9x/NT/2000/XP, or Linux

Mac:
200Mhz+ PowerPC processor
64MB RAM
100 MB Hard drive space (over 1GB hard drive space recommended for storing EClasss)
Mac OS X 10.2 or higher

For Internet publishing, you will need:
A web host, running either Windows 2000, Linux, Unix or FreeBSD.

For CD publishing, you will need:
A CD or DVD-Recordable/Rewritable drive, along with CD/DVD-burning software.

Note that if you use multimedia formats in your EClass, both your machine and the end user's machines need to have the proper viewers installed.
