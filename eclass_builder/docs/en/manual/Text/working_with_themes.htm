You can use the Theme Manager to create, remove, and share EClass themes. (The ability to edit themes from within EClass.Builder is planned for a future release.)  Themes are stored in the "themes" subdirectory of your EClass.

Creating a New Theme

To create a new theme, simply click the "+" button and type in a name for your new theme. The new theme will be based upon the "Default (No Frames)" EClass theme.

Copying a Theme

If you want to modify an existing theme, you can use the Copy button (two papers) to  make a copy of an existing theme. Simply select the theme you wish to copy, and click on the Copy button, then type a name for your new theme.

Removing a Theme

You can remove a theme by selecting the "-" button. Please make sure you don't have any EClasses that depend on the theme before removing it, as it will permanently remove the theme.

Sharing Themes

You can Import and Export EClass themes, so that people can develop their own and share them with others.

To Export a theme, click on the "Export Theme" button and select a folder and filename to save your theme to. You can then send this files to others and share it however you like.

To Import a theme, click on the "Import Theme" button and select the theme file you wish to import. That's it. =)

(Technical Note: The file format for themes is simply the ZIP format, so by renaming the file you could open a theme archive with any Zip software.)

Editing Themes

Themes are stored in the "themes" subdirectory of the EClass.Builder program directory. Click on the folder of the theme you wish to edit, and then open the file "default.tpl" in any text editor.
