Why EClass.Builder?

There are a large number of other software packages out there for building learning modules, documentation or web sites, so what's so great about EClass.Builder?
  • Easy to use interface and short learning curve - if you've worked with a word processor, you know how to use EClass.Builder
  • Clean up HTML tool (using tidylib) makes sure your documents adhere to the latest standards and removes non-standard elements added by tools like Microsoft Word
  • "Open source", which means it is free to use, modify and redistribute
  • Run on Windows, Mac and Linux
