eBook Builder Documentation


eBook Builder is a full-featured software package for developing eBooks, learning modules, and other forms of electronic documentation. The aim of this program is to reduce the technology learning curve associated with creating online educational content. It runs on Windows, Linux and Mac OS X, and documentation built using eBook Builder can be deployed to eBook readers, Learning Management Systems, the Internet or CD-ROM.

What's New?

The major new features of version 3.0 are:
  • Inline Document Editing: Instead of converting documents from MS Word or other formats to HTML and importing them to eBook, now you can just create a new document and start typing! You can also copy and paste text and other content from almost any other program.
  • ePub Standards Support: Export your eBook to a standards compliant format viewable by almost all eBook readers.
