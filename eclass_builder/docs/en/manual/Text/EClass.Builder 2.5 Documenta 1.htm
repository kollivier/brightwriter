EClass.Builder Documentation


EClass.Builder is a full-featured software package for developing EClasss, learning modules, and other forms of electronic documentation. The aim of this program is to reduce the technology learning curve associated with creating online educational content. It runs on Windows, Linux and Mac OS X, and documentation built using EClass.Builder can be deployed to EClass readers, Learning Management Systems, the Internet or CD-ROM.

What's New?

The major new features of version 3.0 are:
  • Inline Document Editing: Instead of converting documents from MS Word or other formats to HTML and importing them to EClass, now you can just create a new document and start typing! You can also copy and paste text and other content from almost any other program.
  • ePub Standards Support: Export your EClass to a standards compliant format viewable by almost all EClass readers.
