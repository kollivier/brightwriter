Graphics Production & Editing

When creating an eBook, you will often need to incorporate graphics to help visualize concepts or drive home a point. To do this, you would use either a vector-based or a raster-based drawing program.

Vector-based Drawing Programs

Vector-based drawing programs are used when you're working with shapes or sketch art - graphs, pie charts, simple drawings. It is called vector because the drawing, to the computer, is simply a series of lines and shapes with properties such as color, thickness and transparency. Similar to 'vectors' in the math world. Drawing using a vector-based tool is much like drawing with a pencil, you can easily erase or adjust each piece of the picture. 

Some popular vector-based drawing programs include:
  • Adobe Illustrator,  the de-facto standard industry tool for professional drawing and illustration. 
  • OpenOffice.org Draw, a vector-based drawing extension for the OpenOffice.org office suite.
  • SVG-Edit, a cross-platform, browser-based vector drawing program.
Although you would save the graphics in a vector-based format while you are working on them, when you export the graphics to include in your EClass, they will need to be in a raster-based format, either GIF, JPEG or PNG. Please see the software's documentation for instructions on how to do this.

Raster-based Drawing Programs

In contrast to vector-based drawing programs, raster-based programs are better suited for creating and editing more complex graphics, like photographs. They store their data as pixels, not lines and shapes, with each pixel having color information stored within it. It is like drawing with a pen - any change you make is permanent once you save and close the document. It is much like drawing with a marker or pen. If you open it later, you cannot simply delete the drawing you did, you'd need to draw over it, or erase it entirely.

Some popular raster-based drawing programs include:
  • Adobe Photoshop, the de-facto standard industry tool for professional raster-based drawing and photo manipulation.
  • The GIMP, a free and open source graphics manipulation program that has a similar level of functionality to Adobe Photoshop.
As with the vector-based programs, when exporting the graphics for inclusion in your EClass, you will need to save them in either GIF, JPEG or PNG format. Please see the software's documentation for more instructions on how to do this.
