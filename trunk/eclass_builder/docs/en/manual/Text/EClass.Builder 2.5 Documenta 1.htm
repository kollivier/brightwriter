EClass.Builder Documentation


EClass.Builder is a full-featured software package for developing EClasss, learning modules, and other forms of electronic documentation. The aim of this program is to reduce the technology learning curve associated with creating online educational content. It runs on Windows, Linux and Mac OS X, and documentation built using EClass.Builder can be deployed to eBook readers with ePub support, Learning Management Systems, the Internet or CD-ROM.

What's New?

The major new features of version 3.0 are:
  • Inline Page Editing: Instead of converting documents from MS Word or other formats to HTML and importing them to EClass, now you can just create a new document and start typing! You can also copy and paste text and other content from almost any other program.'
  • HTML "Clean Up" Support: Standards like ePub require the XHTML content to comply with strict open standards. Programs like Microsoft Word are notorious for exporting non-standard HTML which will not be readable by ePub viewers or some other programs. So now EClass has a feature to automatically convert this HTML to standards-compliant XHTML, and generates it by default when you edit.
  • ePub Standards Support: Export your EClass to a standards compliant format viewable by almost all eBook readers.
  • IMS Package import / export: Export and distribute your EClass project as an IMS Content Package ZIP file. Or import IMS Content Package files from other sources, such as MIT OpenCourseWare. 
  • Less Clutter: In previous versions, EClass would put CD and web export files into the project directory. Now, you specify an export directory and any files added to the project are placed there. That way, you're not uploading CD files to the web, or including them in your ePub / IMS export.
  • Improved Unicode Support: There have been numerous fixes ensuring that an EClass created in one language can be viewed and edited on other computers, even those which may be set to another language.
