Quiz pages are currently auto-grading and support multiple-choice questions. Click "Add" to create a new multiple-choice question, "Edit" to edit an existing one, and "Remove" to delete a question. Here is the dialog for editing questions:



Simply type in the question up top, then type in each possible answer and check the correct box for the correct answer(s). Click the "Feedback" button to define correct and incorrect feedback in this dialog:



The feedback will be displayed to the right of the question when the end user submits their answers.
