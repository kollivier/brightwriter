System Requirements

To run EClass.Builder and view EClasses, you need the following:

PC:
Intel Pentium III Processor or above
256MB RAM
100 MB Hard drive space (over 1GB hard drive space recommended for storing EClasss)
Windows XP or higher

Mac:
Intel processor
256MB RAM
100 MB Hard drive space (over 1GB hard drive space recommended for storing EClasss)
Mac OS X 10.5 or higher

For Internet publishing, you will need:
A web host with FTP upload support and enough disk space to store your EClass.

For CD publishing, you will need:
A CD or DVD-Recordable/Rewritable drive, along with CD/DVD-burning software.

Note that if you use multimedia formats in your EClass, both your machine and the end user's machines need to have the proper viewers installed and meet the system requirements of that player.
