Creating a New Project

There are several ways to create a new project. When BrightWriter starts, you can simply pick the "New Project" option in the dialog that appears. Or, you can always create a new project by going to File->New Project or clicking on the  button in the toolbar. When you create a new project, the following dialog will appear:



Simply type the name of the project, and optionally a description and keywords for your EClass. Once you are finished, click OK to create the new project! It will create a project folder with this name in your EClass Projects directory.
