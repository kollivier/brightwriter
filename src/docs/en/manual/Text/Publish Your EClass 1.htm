Publish Your EClass

Once you have finished creating and organizing your EClass, it is time to publish it. To publish your EClass, you will need to have access to either a CD-Recordable/Rewritable drive (for CD publishing), or have a web site you can post the EClass to (for Internet publishing). We will cover the steps for both methods below.

Publishing to CD / DVD 
From the main screen, either click on the  button in the toolbar, or select File->Publish->To CD / DVD from the main menu. BrightWriter will now update the full-text index (if you enabled searching) and prepare your EClass for CD / DVD publishing. When that is finished, a folder will appear with all the files that need to be burned to CD / DVD. You will need to either copy and paste, or drag and drop, the files into your CD recording software.

Publishing to Internet/FTP

From the main screen, either click on the  button, or select File->Publish->To Web Site from the main menu. A dialog will appear asking for some information.



Often, most of this information will be given to you by either your network system administrator, or your internet service provider. The Host is the name of the web site you would like to post your EClass to, Username and Password are your user account information, and Directory is the folder on the web site where you want your EClass to reside.

The Passive FTP option is sometimes necessary to enable uploading to some internet service providers. If you are experiencing problems uploading, try selecting the Passive FTP option and trying again.

The dialog will show upload progress for each individual file as well as for the entire upload process as a whole. Note that this may take some time on slow internet connections.
