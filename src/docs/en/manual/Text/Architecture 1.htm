BrightWriter is written entirely in the Python programming language. Here's a high-level overview of the components of the system.

How BrightWriter Works

BrightWriter is similar to a content management program in nature, although geared towards documentation and e-learning. It seeks to enable the user to accomplish three main tasks:
  • Import documentation data from many disparate locations and content formats
  • Edit and manage data via an easy-to-use interface
  • Publish the documentation in several popular formats
