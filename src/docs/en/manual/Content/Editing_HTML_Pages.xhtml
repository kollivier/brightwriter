Editing Pages

When you create a new page in BrightWriter, a new page with the title at the top appears in the editor. You can edit this page using the toolbar up top, or using the menu bar. This section will cover how to perform various tasks in the BrightWriter page editor. 

Text Formatting

You can change text formatting using either the toolbar or the Format menu in the menu bar, and sometimes also with the use of keyboard shortcuts. Here is a list of commands and how to use them:

NOTE: When CMD is written, it refers to the modifier key you press to activate the keyboard shortcut. On Mac, CMD is the Apple key, and on Windows, it is the control key. For menu commands, -> means to select a sub menu. For example, Format -> Text Style -> Bold would mean "Select Bold from the Text Style sub-menu of the Format menu."

Bold: press  in the toolbar, or select Format -> Text Style -> Bold from the menu, or press CMD+B on your keyboard. To remove bold, simply select the bolded text and run this command.

Italic: press  in the toolbar, or select Format -> Text Style -> Italic from the menu, or press CMD+I on your keyboard. To remove italics, simply select the italicized text and run this command.

Underline: press  in the toolbar, or select Format -> Text Style -> Underline from the menu, or press CMD+U on your keyboard. To remove underline, simply select the underline text and run this command.

Increase Indent: press  in the toolbar.

Reduce Indent: press  in the toolbar.

Add bullets: press  in the toolbar.
