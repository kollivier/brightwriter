Working with Images

This section details how to insert a created graphic into your EClass. It does not cover how to create graphics. For more on that, please see the "Multimedia Overview" section.

To add an image, press the  button on the toolbar, or select "Image" from the "Insert" menu. Select the file in the file picker dialog that appears, and click "OK" to insert it. The image will be copied into your project directory. Deleting an image is as simple as selecting it and hitting the Delete key on your keyboard. 

To edit image properties, select the image, right-click, and click on the "Image Properties" menu item. This will show the Image Properties dialog.


Here is a breakdown of the various properties: 
  • Image Location: lets you change the image that is displayed. 
  • Text Description: a text description of the image, this is helpful for those with disabilities or in case the image is unable to display for some reason.
  • Image Alignment: how the image displays around the other content. 
  • Width: the width you want for the image, if other than the full image size.
  • Height: the height you want for the image, if other than the full image size.
Once you are finished making changes, click "OK" to apply the changes.
