Working with Links

To create links in your EClass page, just click on the  button in the toolbar, or select "Hyperlink" from the Insert menu. To edit or remove existing links, right-click on the link, and select either "Link Properties" or "Remove Link", respectively. When you add or edit a link, the Link Properties dialog will appear.



The Link Properties dialog has two fields. One to specify the link itself, and one to specify where the link opens when it is clicked on.

To set the link, you can either type in a web link (starting with http://, like http://www.google.com), or you can click the "Browse" button to link to a file on your computer. Just select the file you wish to link to, and click "Open". If the link is outside of your EClass directory, it will be copied there for you. 

In most cases, you can leave the "Open In" section blank, which applies the default behavior of opening the link in the current window. However, in addition to being able to open the link in the current window, there are several other options:
  • Current Window: the default.
  • New Window: this opens a new browser window for the link.
  • Current Frame: Pretty much the same as Current Window, and the default for pages using frames.
  • Parent Frame: For pages using frames, this opens the link above the frames, replacing all the frames with the new link contents.
If you're not sure which to use, just leave the default. Clicking "OK" creates or updates the link.
