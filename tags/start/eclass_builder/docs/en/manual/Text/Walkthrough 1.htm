In this tutorial, we will step through the process of creating an EClass so that you can see the steps involved. The basic outline of the EClass creation process is as follows:

  1. Create a New Project: Name your project and (optionally) provide some simple metadata.
  2. Choose a Theme for Your Project: Decide on a look and feel for your project using the theme manager.
  3. Add/Edit Pages: To do this, you can import files (for example, Word, PDF, Flash) or you can use EClass' own plugin content formats to create new files, such as a web page, quiz or EClass page.
  4. Publish your EClass: Publish your EClass to CD-ROM or the Internet
